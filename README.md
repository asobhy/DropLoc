# Facebook Clone
### This is a realtime social network that has few facebook functionality. Built with the following technologies mentioned below.

## Backend Technologies:
  * Python
  * Django
  * Websocket
  * redis
  * Json

## Frontend Technologies:
  * Html
  * CSS
  * Javascript
  * Fetch API
  * Websocket
